# Climb League Web

Next.js 16 frontend на TanStack Query, организованный по Feature-Sliced Design. Полная архитектура, env, Docker и команды проверки описаны в [общей инструкции](../README.md).

Быстрый development-запуск:

```powershell
Copy-Item .env.example .env.development
pnpm install
pnpm dev
```

Приложение запускается на `http://localhost:3079`.
